Anomaly Percentage vs Parameter Value

Calculation Formula

For values below minimum:
Anomaly% = ((min - value) / min) × 100
For values above maximum:
Anomaly% = ((value - max) / max) × 100
For values within range:
Anomaly% = 0
* Maximum anomaly score is capped at 100%
